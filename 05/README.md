# Домашняя работа №5
Сериализатор поддерживающий типы bool и uint64\_t.
# Запуск 
Требования для запуска: g++(std=c++17)
```
    1. make
    2. make test
```

Также присутствует test2, который проверяет, что программа не скомпилируется при неправильной структуре для десериализации.
- его можно запустить при помощи make test2.

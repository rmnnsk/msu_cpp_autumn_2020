# Домашняя работа №4
Класс для работы с большими целыми числами. Поддерживаются операции:

оператор вывода в поток;
сложение;
вычитание;
умножение;
унарный минус;
все операции сравнения;
копирующий и перемещающий конструкторы/операторы;

# Запуск 
Требования для запуска: g++(std=c++17)
```
    1. make
    2. make test
```
